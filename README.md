## 💡 About

**Mesto** is a modern single-page application where users can share photos, like posts, and edit profile.  
The project is built as part of a frontend learning program and demonstrates working with **API, forms, validation, and OOP components in JavaScript**.

Users can:
- edit profile information
- update avatar
- add and delete cards
- like/unlike cards
- view images in a popup

## ✨ Features

- User profile editing (name, about, avatar)
- Add and delete photo cards
- Like system with live updates
- Image preview popup
- Form validation
- Modular OOP architecture
- API integration

## 🌐 Live Demo

https://mesto-lemon.vercel.app/

## 🛠 Tech Stack

- **HTML5** – structure  
- **CSS3** – styling (BEM methodology)  
- **JavaScript (ES6+)** – application logic  
- **Vite** – bundler and dev server  
- **REST API** – backend communication
